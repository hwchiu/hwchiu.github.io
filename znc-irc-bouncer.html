ZNC IRC bouncer

Posted on 2014-07-23 Edited on 2023-06-23 Views:
ZNC IRC Bouncer 安裝筆記

Environment

  • FreeBSD 10.0-Release

INSTALL

pkgng

  1. pkg install znc

Ports

  1. cd /usr/ports/irc/znc
  2. make config
  3. make install & clean

Config

znc –makeconf

  • add listen port.
  • add user
  • add network. ex: freenode
    • add irc server. ex: irc.freenode.net
    • you can add the default channel passowd by a key=? option in znc.conf

Usage

Find a irc client

  • AndChat on Android
  • kiwiirc on Web
